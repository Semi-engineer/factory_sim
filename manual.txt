‚Ä¢ Percentage of time machine is actively working
‚Ä¢ Formula: (Working Time / Total Time) √ó 100
‚Ä¢ Target: 70-85% (allows for flexibility)

‚Ä¢ Overall machine performance including quality
‚Ä¢ Factors in defects, downtime, speed losses
‚Ä¢ Formula: (Good Output / Theoretical Maximum) √ó 100

THROUGHPUT:
‚Ä¢ Rate of job completion
‚Ä¢ Measured in jobs/hour or pieces/hour
‚Ä¢ Key indicator of factory productivity

‚Ä¢ World-class metric combining:
  - Availability (uptime percentage)
  - Performance (speed efficiency)
  - Quality (defect-free output)
‚Ä¢ Formula: Availability √ó Performance √ó Quality
‚Ä¢ World-class target: >85%

‚Ä¢ Check Python installation (python --version)
‚Ä¢ Verify virtual environment activation

‚Ä¢ Lower simulation speed

PROBLEM: No jobs flowing
SOLUTION:
‚Ä¢ Verify jobs are in machine queues
‚Ä¢ Check machine status (not broken)
‚Ä¢ Ensure simulation is started
‚Ä¢ Verify job routing

PROBLEM: Charts not updating
SOLUTION:
‚Ä¢ Check Analytics tab is visible
‚Ä¢ Restart simulation
‚Ä¢ Verify data collection is enabled

‚Ä¢ Verify machines are added to line

‚Ä¢ Ensure line layout is applied
‚Ä¢ Verify routing configuration

PROBLEM: Export functions failing

‚Ä¢ Check folder permissions
‚Ä¢ Verify disk space
‚Ä¢ Update file paths
‚Ä¢ Check installed libraries
